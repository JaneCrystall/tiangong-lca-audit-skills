# Asset Boundaries

## Skill 内保留

- 审核执行流程和判断政策。
- 过程、模型、输出、纠偏和平台操作方法。
- 唯一结构化规则源。
- 审核结果、通过报告和纠偏记录模板。
- 按需查询的分类数据。

## Skill 外保留

- 脱敏 API 示例与回归案例：`tests/`。
- 维护者架构、环境和平台接入说明：`docs/`。
- 平台 API 与规则执行代码：`src/`。
- 真实审核案件、source 原文、PDF 抽取文本、截图与平台操作日志：`cases/`。

## 不进入仓库

- 原始审核材料和未脱敏平台数据。
- 账号、token、个人联系方式和签字。
- 来源项目目录清单、长篇资料摘要和重复规则。
- 没有实际功能的占位脚本。
